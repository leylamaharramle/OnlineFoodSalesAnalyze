The dataset contains customer-level information collected from an online food ordering platform.

## Exploratory Data Analysis (EDA)
The following analyses were performed:
- Age distribution analysis
- Gender and marital status distribution
- Occupation and education level frequency
- Average family size
- Location hotspot analysis using pin codes
- Feedback rate calculation
- Correlation analysis between numerical variables

Visualizations include:
- Bar charts
- Pie charts
- Histograms
- Heatmaps

## Data Preprocessing
- Selected relevant features for modeling
- Removed missing values
- Applied **One-Hot Encoding** for categorical variables
- Split data into training and testing sets

## Machine Learning Model
- **Model Used:** Logistic Regression  
- **Learning Type:** Supervised Learning (Classification)
- **Target Variable:** Positive vs Negative outcome  

## Model Evaluation
The model was evaluated using:
- Accuracy
- Precision
- Recall
- F1-score
- Confusion Matrix
